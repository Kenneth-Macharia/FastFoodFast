# The FastFoodFast Web App [API IS DOWN FOR MAINTENANCE]
 
## Introduction
This repository hosts the development of an online app for FastFoodFast, a 5 star restaurant. Users will interact with the website frontend to order food for delivery.

## The Components
The app comprises of:

    1.UI pages developed using HTML5, CSS3 and vanilla JavaSript

    2.A RESTful API developed using Python Flask, persisting to a PostgreSQL database

## Viewing the UI web pages
The pages are hosted on Github pages. Follow this link to interact with them:

    https://kenneth-macharia.github.io/FastFoodFast/
